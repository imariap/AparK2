# AparK2

Proyecto de gestión de un parking con lavadero de coches y taller para la asignatura de Ingenieria del Software

### Patrones utilizados

 - Fachada
 - Servicio de aplicación
 - Transfer
 - MVC
 - Singleton
 - DAO
 - Factoría
 - Adapter
 - Controlador frontal
 - Plantila
